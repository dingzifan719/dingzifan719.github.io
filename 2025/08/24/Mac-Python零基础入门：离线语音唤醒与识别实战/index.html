步骤一：语音唤醒 (Wake Word)

目标：让程序能一直监听，并在听到特定词（如“小电脑”）时激活。

  1. 去 Picovoice官网 注册账号。

  2. 在控制台创建一个免费的唤醒词（比如“小电脑”的中文模型），下载 .ppn 文件。
    注册完成后点击“Create Wake Word”（创建唤醒词）按钮，即可得到最后的.ppn文件。

  3. 安装库：pip install pvporcupine
